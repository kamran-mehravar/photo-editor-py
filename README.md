# Photo Editor (Python Prototype)

An early Python-based version of my Photo Editor project.

## Overview

This repository contains the **initial version** of my Photo Editor project, implemented primarily in **Python**.

It represents an earlier stage of the editor before the project evolved into the newer **WebGL-based Photo Editor** version. The goal of this repository is to preserve the Python prototype and document the initial architecture and experimentation phase of the project.

The more advanced and newer version of this project is available here:

- [Photo-Editor](https://github.com/kamran-mehravar/Photo-Editor)

## Relation to the Newer Project

This repository should be considered:

- the **first version** of the project
- a **Python-based prototype**
- an earlier implementation phase before the migration to a more browser-oriented and WebGL-based architecture

In short:

- **photo-editor-py** → early Python version
- **Photo-Editor** → later WebGL-based version
